# revenuecat-cli

Unofficial RevenueCat CLI for Python

## Installation

```bash
pip install revenuecat-cli
```

## Usage

```bash
revenuecat-cli v1 entitlements grant --api-key <api-key> --app-user-id <app-user-id> --entitlement-id <entitlement-id>
```
