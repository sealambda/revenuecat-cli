# RevenueCat CLI

Unofficial RevenueCat CLI for Python.

## :gear: Installation

```bash
pip install revenuecat-cli
```

## :hammer: Usage

```bash
revenuecat-cli v1 entitlements grant --api-key <api-key> --user-id <user-id> --entitlement-id <entitlement-id>
```
